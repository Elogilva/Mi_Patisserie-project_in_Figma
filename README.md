# Mi Patisserie – Sistema de Cafeteria

Este repositório apresenta o design de interface do **Mi Patisserie**, um sistema de gestão para cafeterias desenvolvido por mim e minha equipe como parte de um projeto acadêmico.  
O design foca em oferecer uma experiência de uso elegante, intuitiva e eficiente, tanto para funcionários quanto para clientes.

## Visão Geral

**Principais objetivos:**
- Simplificar o gerenciamento de vendas e pedidos  
- Oferecer uma interface amigável para funcionários e clientes  
- Suportar as principais funcionalidades do dia a dia de uma cafeteria  

## Destaques do Design
- Layout moderno e minimalista  
- Fluxo de navegação fluido e intuitivo  
- Foco em usabilidade e eficiência  
- Design responsivo, adaptável a diferentes dispositivos

# Mi Patisserie – Coffee Shop System

This repository presents the interface design for **Mi Patisserie**, a coffee shop management system developed by me and my team as part of our academic project.  
The design focuses on an elegant, intuitive, and efficient user experience for both employees and customers.

## Prototype

View the interactive Figma prototype:  
[ Figma (Desktop)](https://www.figma.com/proto/Zj8zTQhCf60nAVLT7AdYcm/Software-de-venda?node-id=681-15&starting-point-node-id=681%3A15)
[Figma (Mobile)](https://www.figma.com/proto/Zj8zTQhCf60nAVLT7AdYcm/Software-de-venda?node-id=660-13&p=f&t=XnLhlAaB4C0COaPQ-0&scaling=scale-down&content-scaling=fixed&page-id=0%3A1&starting-point-node-id=660%3A13)

## Overview

**Main objectives:**
- Simplify sales and order management  
- Provide a user-friendly interface for staff and customers  
- Support essential features for daily coffee shop operations  

## Design Highlights
- Modern and minimalist layout  
- Smooth and intuitive navigation flow  
- Focus on usability and efficiency  
- Responsive design suitable for multiple devices  

## Tools Used
- **Figma** — UI/UX design and prototyping  
- **GitHub** — version control and project documentation
